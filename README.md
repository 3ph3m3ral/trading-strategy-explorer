# Trading Strategy Explorer

Notebooks and scripts to test draft strategies for automatic trading.
Dataset available EOS / BTC 2020-2021 

### How to Run

 1)
```
jupyter-notebook 
```
OR <br>
```
jupyter-lab
```

2) Run the hole notebook
3) Modify the strategy parameters
